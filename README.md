# glow

<img src="img/glow.svg" align="right" width="150">

**The fast terminal image display. Written in Rust.**

![Rust](https://img.shields.io/badge/language-Rust-f74c00) ![License](https://img.shields.io/badge/license-Unlicense-green) ![Platform](https://img.shields.io/badge/platform-Linux%20%7C%20macOS-blue) ![Status](https://img.shields.io/badge/status-planned-yellow) ![Stay Amazing](https://img.shields.io/badge/Stay-Amazing-important)

Terminal image display library supporting kitty graphics protocol, sixel, and w3m. Renders images inline in any TUI application. Used by Scroll, Pointer, and Kastrup. Feature clone of [termpix](https://github.com/isene/termpix) rewritten in Rust for speed and single-binary distribution.

## Status

**Planned.** This project is part of the Fe2O3 Rust Terminal Suite.
